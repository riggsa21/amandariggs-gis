Workflows & Methods

Hydrology & DEM Conditioning Workflow

  1. Resample LiDAR-derived DEM and fill sinks.
  2. Generate flow direction and flow accumulation rasters.
  3. Create a stream raster using threshold-based accumulation.
  4. Burn streams into the DEM using conditional expressions.
  5. Recalculate flow direction/accumulation and extract stream network.
  6. Validate using reference data and overlap/intersection metrics.

Land Cover Classification & Change Detection

  1. Preprocess satellite imagery (stack bands, clip to AOI).
  2. Create training samples and run supervised classification.
  3. Assess accuracy using a confusion matrix and validation points.
  4. Reclassify classes into unified categories for multi-year comparison.
  5. Compute land cover change statistics and create change maps.

Automated Portfolio Layout Export (ArcPy)

  1. Scan a file geodatabase for key feature classes (streams, roads, AOIs).
  2. Build or update a map in ArcGIS Pro using those layers.
  3. Apply a standardized layout template and map frame extent.
  4. Export layouts to PNG/JPEG at web and print resolutions.
